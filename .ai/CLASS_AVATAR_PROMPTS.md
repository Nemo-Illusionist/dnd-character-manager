# Default Class Avatar Prompts (DALL-E 3)

Промпты для генерации дефолтных аватарок персонажей по классам D&D 2024.

## Стиль

- Тёмный фон (#2a2a2a–#3a3a3a) — совпадает с UI
- Фиолетовая подсветка (#7c3aed) — акцентный цвет приложения
- Силуэт без лица — универсальный placeholder
- Круговая композиция — аватарки в приложении круглые (48px–120px)
- Минимализм — modern, clean, function over form

## Default (универсальный)

```
A mysterious dark silhouette portrait of a fantasy adventurer, shown from the shoulders up, facing slightly to the side. The figure has a hooded or helmeted outline with no visible facial features. The background is a deep charcoal gray, almost black. There is a subtle violet-purple rim light (#7c3aed) outlining the edges of the silhouette, creating a soft ambient glow. The style is clean, modern digital art with a matte finish — minimalist and sleek, not painterly or cartoonish. The composition is centered and works perfectly as a circular avatar crop. Dark fantasy mood combined with a contemporary UI design aesthetic, similar to modern gaming tools. No text, no borders, no decorative elements. High contrast between the dark figure and the faint purple glow. Square format, 1024x1024 pixels.
```
