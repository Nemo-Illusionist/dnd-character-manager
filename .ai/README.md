# OmnisGM / One More VTT (OMVTT)

> Название проекта не финализировано

## Что это

PWA-приложение для управления TTRPG (настольными ролевыми играми). Мультисистемный инструмент для мастеров и игроков.

## Основные цели

1. **Для мастера (GM)**: постоянный доступ к актуальным листам персонажей всех игроков
2. **Для игроков**: доступ к shared ресурсам (карты, заметки, объявления) и компендиуму
3. **Для всех**: лёгкий доступ к правилам и данным из компендиума

## Текущий статус

- **Стадия**: MVP в активной разработке
- **Пользователи**: 2 (тестовая группа)
- **Поддерживаемые системы**: D&D 2024 (SRD 5.2.1)

## Использование

- **Игроки**: мобильный/планшет за столом
- **Мастер**: ноутбук
- PWA позволяет установить как приложение на любое устройство

## Коммерциализация

Личный проект с возможной коммерциализацией в будущем.
